7、C语言中的真和假: 0为假,非0为真

8、逻辑运算符：
    （1）逻辑与(&&) : a && b 全真为真,一假为假
    （2）逻辑或(||) : a || b 一真为真,全假为假
    （3）逻辑非(! ) : !a 取反,如果a为真,则结果为假,a为假则结果为真

9、逻辑短路原则:
    （1）逻辑与&&: 左边的表达为假,结果一定为假,则右边的表达式不参与运算
    （2）逻辑或||: 左边的表达式为真,结果一定为真,则右边的表达式不参与运算

10、三目运算符(? :)  的运算过程：
    判断表达式一是否为真,为真则取表达二作为结果,为假则取表达式三作为结果 max = a > b ? a : b

11、a,b,c三个数,要求从键盘输入 找到最大值

    int a,b,c;
    printf("请输入三个数:");
    scanf("%d%d%d",&a,&b,&c);

    int max = a > b ? a : b;
    max = max > c ? max : c;
    printf("最大值为:%d\n",max);

12、数据类型转换：操作符两边的操作数的类型不一致可能发生类型转换。

13、类型转换分为两种：
    （1）隐式类型转换:由编译器自动完成
        float pi = 3.14;
        int a = pi;
        printf("a = %d\n",a);

    （2）显式类型转换
        float pi = 3.14;
        int a = (int)pi;
        printf("a = %d\n",a);
    推荐在写代码时使用显式类型转换
