 ----------------------- 第二天 第三章 数据类型和变量 --------------------------

1、变量的组成： 数据类型(int、float..) + 变量名(a,b,c,r,d,h,w,pi,e)

2、变量的本质： 内存中的存储单元（内存中的一块空间）

3、各种数据类型在内存中占用的字节数
	char — 1字节
	short — 2字节
	int — 4字节
	long — 8字节（在32位编译器下占用4字节）
	float — 4字节
	double — 8字节

4、short、int、long类型的区别： 取值范围不同
   float与double区别： 精确度不同  

5、sizeof(变量名/类型名/值) ： 获取某一数据类型（的变量）在内存中占用的字节数

6、变量名命名规则：
	（1）不能以数字开头，可以由字母、数字、下划线组成
	（2）区分大小写 int A和int a是两个变量
	（3）不能以关键字作为变量名

7、命名的一些规范：
	（1）望文知意 — 一般使用英文单词表示（不要使用中文拼音）
	（2）符合“最小长度-最大信息”
	（3）驼峰命名法（大驼峰、小驼峰）

8、常量：声明后其值不能被修改  const float pi = 3.14;

9、标准输入输出过程：
	 int a;          //编译器分配4个字节 但值是未知的
    scanf("%d",&a); //编译器等待用户从键盘输入一个值给变量a
    printf("a = %d\n",a);//将变量a的值从内存中输出打印在屏幕上

10、printf的两种用法：
    1. 直接输出一个字符串（一句话）
    printf("What a phucking nice day!\n"); // 用printf输出一句话表示今天天气非常好
    
    2. 用于输出变量的值
    int a = 5;
    float f = 3.14;
    printf("a = %d,b = %f\n",a,f);  //占位符

11、scanf在使用时需要注意的两个问题：
	（1）变量名前要加&  如 scanf(“%d%d”,&a,&b);
	（2）双引号中尽量不要加其它额外的修饰，只写格式控制符

    scant输入double类型时的格式控制符为%lf

12、枚举：用一些标签(字符串)来代替了整型值，增加了程序的可读性

13、数组：表示相同类型的值的集合

14、数组元素的访问： 数组名[下标]  下标取值范围：0 ~ n-1
