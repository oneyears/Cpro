1、切换输入法
   Windows系统： ctrl + shift 或 ctrl + space
   苹果(Mac os x)系统： command + space(空格键)

   苹果系统中的快捷键    command -> windows中的ctrl键  
   比如： ctrl + c/v    对应苹果系统中的  command + c/v

2、中英文输入法切换：  shift
